# 🌐 나만의 AI 멀티 채널 리서치 비서

이 프로젝트는 누구나 쉽게 최신 인공지능(LLM)을 활용하여 관심 있는 주제의 뉴스, 논문, 유튜브 영상을 한 번에 수집하고 전문가 수준의 리서치 레포트를 받아볼 수 있는 **통합 AI 리서치 플랫폼**입니다.

## 🚀 바로가기 (배포 링크)
👉 **[AI 리서치 비서 실행하기](여기에_회원님의_스트림릿_앱_주소를_넣어주세요)**

---

## 💡 주요 기능 및 특징

1. **멀티 LLM 지원 (Multi-LLM Architecture)**
   - Google Gemini, OpenAI ChatGPT, Anthropic Claude 등 세계 최고 수준의 AI 메인 엔진을 사용자가 직접 선택하여 리서치를 진행할 수 있습니다.
2. **초고속 병렬 데이터 수집**
   - 구글 뉴스, arXiv 논문, 유튜브 영상, 특정 지정 사이트의 데이터를 비동기 방식으로 순식간에 긁어모읍니다.
3. **주제별 독립 채널 운영**
   - 최대 5개의 관심 주제를 설정하고, 각 주제별로 특화된 URL을 지정하여 맞춤형 모니터링이 가능합니다.
4. **우아한 성능 저하 (Graceful Fallback)**
   - API 키가 없거나 일일 트래픽 한도를 초과하더라도, 시스템이 멈추지 않고 수집된 원문 자료(링크 포함)를 예쁘게 큐레이션하여 사용자에게 제공합니다.

## 📖 사용자 가이드 (어떻게 사용하나요?)

1. **AI 엔진 선택 및 API 키 입력 (선택사항)**
   - 좌측 사이드바에서 원하는 AI 엔진을 선택합니다.
   - 본인의 API 키가 있다면 입력해 주세요. (입력하지 않으면 'AI 요약'은 생략되고 '데이터 수집 결과'만 제공됩니다.)
2. **수집 옵션 조절**
   - 뉴스, 논문, 유튜브 영상 등의 수집 개수를 슬라이더로 조절합니다.
3. **주제(채널) 활성화**
   - 화면 중앙의 채널 탭에서 스위치를 'ON'으로 켜고 관심 있는 검색어(예: 인공지능, 테슬라, 금리 인상)를 입력합니다.
4. **리포트 일괄 생성**
   - 빨간색 `🚀 선택한 활성 채널 레포트 일괄 생성하기` 버튼을 누르면 끝!
   - 1~2분 뒤 나만의 멋진 리서치 레포트가 화면에 출력되며, 마크다운(.md) 파일로 다운로드할 수도 있습니다.
